Frontend Development

Frontend Development is the Graphical_user_interface of a website,through the use of HTML ,CSS and javaScript, so that users can view and interact with that website.

  • HTML: The HyperText Markup Language or HTML is the standard markup language for document designed to be displayed in a web browser.

  • CSS: Cascading Style Sheets (CSS) is a Style sheet language used for describing the presentation of a document written in a markup language such as HTML or XML.



    1. Pure CSS
    2. Bootstrap(framework)
    3. Tailwind(Framework)
    4. Saas(Preprocessor)

  • JavaScript: JavaScript often abbrevaited JS, is a programming language that is one of the core technologies of the World Wide Web , alongside HTML and CSS. It is use to add functionality in website.



    1. Vanilla JS
    2. jQuery(Library)
    3. React(Framework)
    4. Angular(FrameWork)
    5. Vue(FrameWork)
    6. Typescript(Preprocessor)
